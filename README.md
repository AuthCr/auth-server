# credid-server

Credid: Authentication and permission management server.

## Installation

You need crystal and shards installed, on the lastest version.

    make

## Usage

    ./credid-server --ip 127.0.0.1 --port 8999

### Protocol

See the [RFC](https://credid.sceptique.eu/)
